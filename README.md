# Koa-CLI

A Koa Cli, easy to use.

## Installation

```bash
npm i -g koa-cka
```

## Usage

Create a Koa app in directory `my-app`:
```bash
koa create my-app
```

After the project created, you can run:
```bash
cd my-app
npm start
```

To start a dev server:
```bash
npm run serve
```
Once file changes server will auto restart.

Enjoy your coding time.
